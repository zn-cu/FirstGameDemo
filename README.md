# Fox Platformer Demo v6

本版重点修复：

- 重新制作 `tiles.png`，包含草地、土块、空中平台、金币、地刺、花草、指示牌等，风格参考 UI 图右下角的 Tilemap 区域。
- 重做关卡平台高度：角色最大跳高约 128px，所有台阶垂直高度差都控制在 96px 内，可以跳上去。
- 角色尺寸缩小，脚底锚点改为与碰撞盒底部对齐，减少悬浮感。
- 地刺加入碰撞，脚部碰到会扣 1 滴血。
- 掉坑扣 1 滴血，3 滴血用完游戏结束。

## 运行

```bash
npm install
npm run dev
```

## 操作

- A / D 或 ← / →：移动
- W / 空格 / ↑：跳跃
- S / ↓：下蹲；站在空中平台时按下可穿透下落
- R：重开
