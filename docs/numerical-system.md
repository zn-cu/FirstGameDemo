# 数值系统

## 伤害公式

物理伤害：

```text
最终伤害 = 攻击力 - 防御力
```

魔法伤害：

```text
最终伤害 = 魔法攻击力 - 魔法防御力
```

代码中为了避免出现 0 伤害导致敌人或玩家无法被击败，最终伤害最低为 1。
